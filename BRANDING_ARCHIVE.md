# Regulix Original Branding Archive

**Date Archived:** 2025-11-09

## Brand Identity

**Name:** Regulix  
**Tagline:** Regulatory Intelligence  
**Positioning:** Modern regulatory compliance platform powered by AI

## Visual Identity

### Logo
- **Type:** Shield-based logo with robotic/tech elements
- **Symbol:** Blue-green gradient shield representing trust, protection, and intelligence
- **Style:** Modern, geometric, tech-forward
- **Icon:** `<RoboticShieldLogo />` component
- **Colors:** Primary blue-green with gradient effects

### Color Palette

**Primary Colors:**
- Primary Blue: `hsl(221.2 83.2% 53.3%)` - Trust, intelligence
- Primary Foreground: `hsl(210 40% 98%)` - High contrast text
- Accent Blue-Green: Gradient variations

**Secondary Colors:**
- Background: `hsl(222.2 84% 4.9%)` (Dark mode)
- Foreground: `hsl(210 40% 98%)`
- Card: `hsl(222.2 84% 4.9%)`
- Muted: `hsl(217.2 32.6% 17.5%)`

**Semantic Colors:**
- Success/Positive: Green tones
- Warning: Yellow/amber tones
- Error/Destructive: Red tones
- Info: Blue tones

### Typography

**Font Stack:**
- System fonts with fallbacks
- Sans-serif primary: `-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto`
- Clean, modern, highly readable
- Font weights: 400 (regular), 500 (medium), 600 (semibold), 700 (bold)

**Hierarchy:**
- H1: 2.5rem (40px), bold
- H2: 2rem (32px), semibold
- H3: 1.5rem (24px), semibold
- Body: 1rem (16px), regular
- Small: 0.875rem (14px), regular

### Design System

**Style:**
- Modern, clean interface
- Card-based layouts
- Subtle shadows and depth
- Rounded corners (0.5rem default)
- Gradient accents on key elements
- Professional, corporate aesthetic

**Spacing:**
- Consistent 8px base grid
- Generous whitespace
- Balanced layout proportions

**Components:**
- Shield logo as primary brand mark
- Blue-green gradients on hero sections
- Card-based information architecture
- Icon-first navigation
- Professional data visualization

## Notes

This branding established Regulix as a serious, enterprise-grade regulatory intelligence platform with strong emphasis on protection, trust, and AI-powered intelligence. The shield motif and blue-green color scheme created a professional, authoritative presence suitable for corporate compliance environments.
